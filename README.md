## ⚙️ تنظیمات قبل از راه‌اندازی

### ۱. توکن ربات
در `bot.py` خط زیر را پیدا کن:
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```
توکن جدیدت از @BotFather رو جایگزین کن.

### ۲. آیدی ادمین
در `handlers/admin.py` خط زیر را پیدا کن:
```python
ADMIN_IDS = [123456789]
```
آیدی عددی تلگرامت رو بذار (از @userinfobot بگیر).

### ۳. فایل نقشه
فایل `map.webp` رو کنار `bot.py` قرار بده.

## 🚀 راه‌اندازی روی Railway (رایگان)

1. برو [railway.app](https://railway.app) و با GitHub وارد شو
2. یه repo جدید در GitHub بساز و همه فایل‌ها رو آپلود کن
3. در Railway روی **New Project** → **Deploy from GitHub** کلیک کن
4. repo ات رو انتخاب کن
5. در تب **Variables** یه متغیر اضافه کن:
   - Key: `BOT_TOKEN` | Value: توکنت
6. در تب **Settings** → **Start Command** بنویس:
   ```
   python bot.py
   ```
7. Deploy کن! ✅

## 👑 دستورات ادمین

| دستور | توضیح | مثال |
|-------|-------|------|
| `/admin` | باز کردن پنل ادمین | `/admin` |
| `/set <id> <field> <key> <value>` | تنظیم مقدار دقیق | `/set 123 resources سکه 9999` |
| `/add <id> <key> <amount>` | افزودن منبع | `/add 123 سکه 5000` |

**فیلدها:** `resources` | `military` | `navy` | `siege`
